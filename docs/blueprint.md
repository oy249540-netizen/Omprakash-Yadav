# **App Name**: OmPath Math Academy

## Core Features:

- Grade Syllabus Explorer: Comprehensive digital navigation of specialized mathematics curricula for middle school Classes 5 through 8.
- AI Concept Sculptor: An AI-powered tutoring tool that breaks down geometric and algebraic problems into simplified logical steps using conceptual reasoning.
- Direct Admission Gateway: Interactive enrollment form designed to capture student data and bridge direct contact through integrated WhatsApp protocols.
- Live Session Dashboard: A centralized hub providing real-time schedules and direct access links for instructional YouTube streams.
- Educational Resource Vault: Categorized secure repository for downloading practice assignments, test papers, and conceptual PDF guides.
- Parent Support Interface: Floating high-visibility action module for instant academic inquiries and faculty coordination via mobile messaging.
- Trial Session Manager: Simplified registration workflow for reserving 2-day complimentary demo classes for new students.

## Style Guidelines:

- Primary Deep Navy (#1A2D66) chosen for a sense of trust, authority, and academic discipline.
- Background Mist Blue (#F5F7F9) providing a clean, distraction-free environment for learning and reading.
- Accent Cerulean (#1A80E6) to highlight progress tracking and primary call-to-action buttons.
- Recommended font: 'Nunito' (sans-serif), a user-specified choice known for its approachability and clarity. Note: currently only Google Fonts are supported.
- Linear, minimalist icon set with soft rounding to evoke a modern educational app aesthetic.
- Responsive card-based system with intentional white space and focus on mobile-first accessibility.
- Micro-interactions using subtle entrance transitions to guide the student's attention through learning paths.
